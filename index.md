# Sounding the Castle

Sounding the Castle is a digital humanities project documenting musical objects associated with the Hearst Castle collection in San Simeon, California.

![California Early Music Sources logo]({{ "/images/california-early-music-logo.svg" | relative_url }}){: style="max-width: 520px; width: 100%; height: auto; margin: 0.75rem 0 1.25rem;" }

![Ornate castle room with manuscript lampshades and carved arches]({{ "/images/splash-manuscript.png" | relative_url }}){: style="width: 100%; height: auto; border-radius: 6px; margin: 1rem 0 1.5rem;" }

## Collections

- Antiphoners
- Chant fragments/Lampshades
- Instruments
- Piano rolls
- Performances
