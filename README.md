# Sounding the Castle

A digital inventory documenting antiphoners, chant fragments, instruments,
piano rolls, and performances.

This project explores the musical history of the Castle collection through
manuscripts, objects, and performance.

## Collections

- Antiphoners
- Chant fragments
- Instruments
- Piano rolls
- Performances

## Project Status

Work in progress.

## Local Development

This site is built with Jekyll and the Just the Docs theme.

```sh
bundle install
bundle exec jekyll serve
```

The local site runs at `http://127.0.0.1:4000/Sounding-the-Castle/`.

## GitHub Pages

This repository uses a GitHub Actions workflow to deploy the Jekyll site so
GitHub can install the Just the Docs theme from the `Gemfile`.

In the repository settings, set **Pages > Build and deployment > Source** to
**GitHub Actions**.
